# Pre-Pro Paperwork

A collection of standalone browser-based tools for entertainment lighting pre-production. No server required — open any `index.html` directly in Chrome.

## Tools

### Rack Builder (`rack-builder/`)
Visual rack layout tool for planning DMX node, splitter, and breakout configurations.
- Drag-and-drop rack population
- Per-port universe assignments
- Snake / loom connection mapping
- Port-to-port cross-rack connections
- Print-ready layout export

### Loom Builder (`loom-builder/`)
Loom list planning and cable tracking tool.

## Reference Docs (`reference-docs/`)
Manufacturer specs and manuals for common equipment used in show builds.

## Usage
Open `rack-builder/index.html` or `loom-builder/index.html` in a browser. All project data is saved automatically to `localStorage` in your browser.
